Cheatsheet

HTML

Common tags

Tag Name Description 
<!DOCTYPE html>
 Document Type Used to inform the browser about the version of HTML being used. Found at first line in source. 
<head>
 - Section in HTML document used to provide metadata like titles, stylesheets and other info about the page. 
<body>
 - Contains the actual HTML elements and content that the user interacts with. 
<h1> - <h6>
 - Header text used to provide titles for pages or sections/articles. Higher number equals smaller fontsize. 
<embed>
 - Places external content like a picture, video or audio source within the webpage. Does not care about the type of content provided. 
<table>
 - Contains the actual HTML elements and content that the user interacts with. 
<a>
 Anchor Defines a hyperlink, used to link from one page to another. Most important attribute is href.

Common attributes

Tag Name Description 
class
 - Specifies which classnames in a style sheet the HTML element is subject to. 
href
 Hypertext Reference Used to define the destination of the hyperlink, commonly used in a, link tags. 
src
 Source Used to define the location of the external resource. Used with audio, embed, iframe, img, input, script, source, track, video tags 
id
 Identifier Specifies a unique ID for an HTML element. The given ID to any element must be globally unique within the document. 
type
 - Specifies a variation of a button or input element. For embed, link, object, script, source, style tags it specifies the media type. 
alt
 Alternate Specifies an alternate text for an image if the image cannot be displayed. 
target
 - Specifies where to open the linked document or resource.

Semantic tags

Tag Name Description 
<header>
 - Usually contains introductionary content or a navbar to help the user around the page. 
<main>
 - Contains the main content of a given page. Content within should be unique to the document. The main element cannot be a descendant of other semantic tags. 
<footer>
 - Usually contains copyright or authorship information and where to contact the page owner alongside other useful links. 
<section>
 - Defines a section within a document, improving SEO and accessibility to screenreaders. 
<article>
 - The article tag is used for self-contained independent content like a forum post or news story. 
<aside>
 - The aside tag is used for content aside from the content it is placed in, like a sidebar. Has no special properties.

CSS

Text & Elements

Property Value Description 
font-family
 "Arial", serif, sans-serif The font being used for a given HTML element, if given font can't be found browser will use a websafe font. 
font-size
 10px, large, 150% Determines the size of text of the element. 
text-align
 left, center, right Determines the alignment of the text within it's parent element 
color
 #FFFFFF, HSL(a), RGB(a) Determines the foreground color of the element. 
background-color
 #FFFFFF, HSL(a), RGB(a) Determines the background color of the element. 
position
 static, relative Used to inform the browser about the version of HTML being used.
